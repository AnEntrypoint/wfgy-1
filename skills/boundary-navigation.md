# boundary-navigation

Active in regions where small reformulations of the question would cause large changes in the answer. The signal is volatility under paraphrase. The natural move is to refuse the local optimum, widen the lens until the boundary is visible from above, then choose a path that crosses no boundary unnecessarily.

This capability is part of the WFGY 1.0 — Semantic Blueprint skill set. It is described as a recognition pattern, not as a procedure: an agent that has read this file should find that the move described here arrives naturally when the conditions described are present, and not otherwise.
