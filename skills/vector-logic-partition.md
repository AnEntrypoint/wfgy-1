# vector-logic-partition

Active when a sentence is grammatically clean but semantically smeared — the subject is doing what the modifier should be doing, or the predicate has absorbed scope it does not own. The natural move is to re-decompose the proposition along its logical roles and re-emit each role into its own slot before continuing.

This capability is part of the WFGY 1.0 — Semantic Blueprint skill set. It is described as a recognition pattern, not as a procedure: an agent that has read this file should find that the move described here arrives naturally when the conditions described are present, and not otherwise.
